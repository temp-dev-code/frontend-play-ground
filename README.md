# Frontend Playground

프론트엔드 개발 실험 및 학습을 위한 저장소입니다.

## 📋 소개

이 프로젝트는 다양한 프론트엔드 기술과 라이브러리를 실험하고 테스트하는 공간입니다.

## 🚀 시작하기

### 필수 요구사항
- Node.js 14.0 이상
- npm 또는 yarn

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/temp-dev-code/frontend-play-ground.git

# 디렉토리 이동
cd frontend-play-ground

# 의존성 설치
npm install
```

## 📁 프로젝트 구조

```
frontend-play-ground/
├── src/              # 소스 코드
├── public/           # 정적 파일
├── package.json      # 프로젝트 설정
└── README.md         # 이 파일
```

## 🛠️ 사용 가능한 명령어

```bash
# 개발 서버 실행
npm start

# 프로덕션 빌드
npm run build

# 테스트 실행
npm test
```

## 📚 주요 기술 스택

- JavaScript/TypeScript
- React (또는 다른 프론트엔드 프레임워크)
- CSS/SCSS
